一年存一份快照——2018年总结与盘点

2018-12-31

搜索时，遇见失效的网页，想看看不到怎么办？有一个小技巧，点击搜索页上的快照，来查看服务器以前存下的网页。这就是——快照。它不同于备份，备份可以还原，快照只是记录，并不求完整。

2017 年，偷了个懒，什么总结都没写，脑海里留下的关键词只有两个：装修、住院拔智齿。转眼2018 年即将结束，吸取教训，为今年存一份快照。

下厨

这一年，一个大变化是开始享受做菜的乐趣。以前，在豆瓣有过一个中二的签名，叫「土豆丝炒不熟」。因为二十多年来很少有做菜的机会，偶尔做了一次炒土豆丝还是生的。如今新家装修了一年入住，有了条件和心情，就踏上了料理之路。人在吃这件事上是非常平等的。每个人，一日三餐，份量相差不大，即使是山珍海味，仍囿于胃的容量和新陈代谢的速率。而菜的味道，取决于对吃这件事的追求。目前处于刚出道的阶段，在田螺姑娘、小高姐的魔法厨房、cbvivi 的平凡料理、catson 的阿开好菜中贪婪地进修，收获颇丰。下面开始报得意作菜名：香菇鸡腿焖饭、番茄土豆炖牛肉、花菜炒排骨、红烧鸡爪、烤鸡翅、烤花菜、蒸温州鱼饼、蒸腊肉。明年再接再厉，继续享受做和吃。至于洗，就不用操心了，家有幸福感神器——大洗碗机。（得意脸）

播客

喜欢做饭的另一个原因，其实是多了一个听播客的时间。播客是自己的一项重要休闲及学习来源，也是独处时候的最佳排遣。今年听了很多期，想推荐的有：

  • 太医来了 119 期：何为虐童？——从虐童事件看幼儿教育
  • 忽左忽右 7 期：移民运动中的「歧视链」
  • 忽左忽右 28 期：顶级风险投资机构的日常
  • 忽左忽右 32 期：我所亲历的「代表亚洲提问」事件（二十一世纪初的财经故事精彩万分）
  • 纽约文化沙龙 113 期：谈判没有那么难，换一种视角看谈判（谈判过程拥有 Plan B 才能掌握主动权）
  • 疯投圈：自由职业是生活的理想状态吗？
  • 日谈公园 147 期：小史欧洲任我行 德国篇（原来旅游可以这么深度）
  • 一席：手语律师（聋哑人的司法现状是如此糟糕）
  • 大内密谈 609 期：愿我们终囿于厨房与爱
  • 得意忘形 48 期：第一季番外篇之「Know Your Game 啊朋友」
  • 得意忘形 49 期：第一季番外篇之「请管好你的企鹅」
  • 故事FM 129 期：你想象过自己的临终吗？（关于生前遗嘱的「五个愿望」）
  • 故事FM 143 期：逃离上海后，我选择去巴西做一个快乐的穷人
  • 翻转电台：东西方等级社会差异（基于费孝通的乡土中国聊东西方的人际关系和阶级问题）

小店

时间繁忙时，懒得下厨，就下馆子改善伙食，尽可能在「钞票粉碎机」来临之前享受一下自由。在宁波这三年，有几家印象深刻的小店：

  • 鸟町：烤青花鱼、肥牛饭
  • 八道喜：关东鳗鱼饭、关西鳗鱼饭
  • 丸龟制面：番茄猪软骨饭、烤青花鱼
  • 东鼓道地下二楼美食广场：三鲜面、三鲜炒粉干、糯米饭
  • 渔人码头：夏天吹着江边的风吃大排档

看书

每年定计划的时候，看书总少不了，就是数量不达标，今年看了几本不错的。

最近看的一本是《邓小平时代》。书中详细记录了邓小平在成为领导核心之前经历了怎样的「三起三落」。文化大革命当中，高层是什么样一个状况。同为文革的受害者，如何在拨乱反正时，不使社会动乱走向另一个极端。他又是如何通过没有加冕礼的权利交接后，纵横捭阖，开启了一个新时代。今年是改革开放40周年，倒也应景。

《江海寄余生》，则是一位经济专业人士的新浪博文集。从书中可以看到他风险意识强、重视抉择能力、家庭观念重等特点。文字功底好，思路清晰，对个人有所启发，还为此专门写了篇一位澳洲移民博文集的非官方序的博客。

《指数基金投资指南》，是指数基金的入门指南，包含了一些炒股的基本知识，科普读物。同样写过总结。

米什金著的《货币金融学》，是一本金融学的教材。要学习经济与金融，建立知识架构，科班教材是一个很好的学习路线。宏观经济学、微观经济学都扫过一遍，2018年看了货币金融学，介绍了金融的运作机制。开篇谈到个人与个人之间借钱还钱的交易过程存在信息不对称的问题。交易前，由于不了解各个借款人的情况，光凭利率选择最终债务人，容易劣币驱逐良币，引起逆向选择的问题，因为越需要钱的人越容易接受高利率，却也越有可能还不起。在交易后债务人有从事高风险高收益投资的道德风险，因此银行这个媒介有了存在的必要。这只是一个小例子，书中还有很多并不深涩的知识点。当然，书中还有部分数学的推理选择了跳过，减少看书的阻力。

同样，斯蒂芬罗斯所著的《公司理财》，也属于经济学范畴的著作，从中可以学习如何看公司的财务报表、公司发行股票和债券的区别以及其他开公司的需要注意的财务知识。并没有书名看起来那样艰深无聊，反而蛮有趣。

工作

工作上，进一步学习 JavaWeb 的代码知识，按着 Github 一个项目拆解、再模仿搭建 SSM 项目，并做 Markdown 笔记利于回忆；学习了 SoupUI 的简易用法，与第三方对接接口和调试接口的时候细节更加明确，提高沟通效率，掌握主动权；提高数据库 SQL 的编写能力；增加 Axure 绘制高保真原型的能力；总体，对于自我的成长还算满意，学习强度还需加强。来年，对自己提个要求，希望对智能工厂的业务及技术做更多的输出，达到专家级别。

糟糕的事

此外，大环境中，仍有很多糟糕的事在发生。

  • 贸易战
  • 中国发生非洲猪瘟，近几年估计都不会断绝
  • 长春长生假疫苗
  • 乐清滴滴顺风车杀人
  • 重庆无赖乘客抢方向盘，全车坠入江底无一生还
  • 乐清走失儿童闹剧
  • 最高院法官为防不测，流出重要案件的自述视频

当下，人人都有智能手机，雁过也不一定无痕，一翻照片或许就能知道某年某月几点几分有飞鸟路过。我个人的 2018 年小记，也算这个时代的小注脚。
