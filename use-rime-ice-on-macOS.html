macOS下Rime输入法配置雾凇拼音

2026-02-12

在 macOS 下，搜狗输入法由于隐私问题没有使用，自带的输入法在特殊符号等方面体验仍然不佳，Rime 是一个开源免费跨平台的解决方案，在 macOS 下使用鼠须管作为客户端，有高度的自定义性，支持自定义词库、自定义键盘按键映射、自定义皮肤、根据字频调整候选顺序。

然而，默认的鼠须管并不好用，一打开就是各种不知名的繁体，劝退。好在有各种优化方案，比如雾凇拼音，只要微调就能符合个人的使用习惯，非常棒！

安装鼠须管

  1. 在官网下载安装 Squirrel。
  2. 在系统输入法中增加鼠须管。

配置雾凇拼音

  1. 到 ~/Library 路径下删除安装时默认生成的 Rime 文件夹，从 Github 上下载雾凇拼音配置，可以用以下命令下载到一个名为Rime的文件夹，也可以手动下载解压。
git clone https://github.com/iDvel/rime-ice.git Rime --depth 1
  1. 在输入法菜单栏点击「重新部署」。

使用技巧

  1. 补充 custom_phrase.txt，形成自定义词汇库，比如把我常用的邮箱用缩写映射，常用的软件名词大小写修正。
  2. vfh，可以调出符号，v1 可以调出数字 1 的各种形式（1️⃣、①、壹、Ⅰ），配置写在 symbols_v.yaml,要调整数据可以直接改这个配置文件。
  3. 修改 default.yaml 文件，可以修改按键映射。我为了在 markdown 模式下输入代码片段方便，把以下这行配置注释了。
'`' : '·'
  1. squirrel.yaml，可以修改主题、字体，我选择使用了方正屏显雅宋。
  2. 更多技巧可以看官方文档。
