查看网站是否支持 RSS 方法

2019-09-26

习惯用 RSS 订阅博客，新文章发布后可以第一时间收到通知。

部分网站没有在页面上展示 RSS 链接，但是实际上是支持的，可在 Chrome 右键，查看网页源代码，在 HTML 头部包含 type 为 application/atom+xml 或 application/rss+xml 的 link 元素来指明 RSS 链接。
