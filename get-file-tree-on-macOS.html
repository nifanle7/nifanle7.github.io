macOS终端生成文件树方法

2023-01-11

在 Github 上浏览项目，或者观看一些开发教程时，会看到字符格式的文件夹目录结构，很清晰。方法简单的，使用 tree 命令就能实现。

第一步，打开终端，使用 Homebrew 安装 tree。

brew install tree

第二步，终端切换到指定文件下，使用 tree 命令，如果加了 -L 1 的参数，代表只展示一级目录，不加参数默认展示所有文件列表。

cd Desktop
tree -L 1

效果如下：

.
├── db.json
├── node_modules
├── package.json
├── public
├── scaffolds
├── source
└── themes
