Weekly | 明朝有封事，数问夜如何

2022-07-17

一首诗

新买的 Boox Poke4S 屏保可以设为诗词，最近在上面看到一首杜甫的诗，挺喜欢的。

春宿左省
花隐掖垣暮，啾啾栖鸟过。
星临万户动，月傍九霄多。
不寝听金钥，因风想玉珂。
明朝有封事，数问夜如何。

好的诗可以传递情绪。想象一下，自己明天有一场重要的答辩，在紧张的准备之时，看到傍晚落霞余晖，栖鸟飞过，心生忐忑。随着夜晚到来，斗转星移，月色清凉，时睡时醒，迷迷糊糊问自己好几遍现在是晚上几点，为什么这个夜晚格外的长。

一个工具

Raycast 是一款免费的启动器，颜值相比 Alfred 更加现代，与 macOS 系统功能及 app 软件结合较好。所以直接使用默认快捷键 Option + Space 替换了 Spotlight。

我最喜欢自带的窗口管理功能，使用 Rectangle 的预设快捷键。

主要用到的快捷键如下：

  • Option + Command + ←：窗口缩放到屏幕二分之一，靠左。
  • Option + Command + →：窗口缩放到屏幕二分之一，靠右。
