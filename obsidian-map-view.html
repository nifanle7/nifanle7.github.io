Obsidian 在地图上做笔记

2022-05-27

使用场景

好些年前，我有过这样一个设想：要是能在高德地图、百度地图 GIS 系统上给各个地点写笔记该多好，可以应用于个人游记、美食地图、旅游攻略、心愿清单、出行足迹等多种场景。如今 Github 上发布了一款 Obsidian 的地图插件，结合第三方的 GIS 源，完全可以打造私人地图卡片。

使用方法

1、安装地图插件。插件名为： obsidian-map-view 。由于插件暂未上架官方插件市场，可以到 Github 上下载到本地的插件文件夹，或者可以通过 BART 安装。

2、在设置中启用插件。

3、在边栏点击小地球图标，打开 Map View。通过地图选点或者搜索「Search Active Map View」，右键点击选择生成带地图链接的笔记。

4、生成完之后，可以在地图上点击弹出笔记详情。也可以在笔记的坐标下右键点击在地图中显示。

5、地图的卡片支持标签过滤，比如可以按美食类、足迹类、计划类分别标记，当只需要看其中一类时，可以屏蔽其他的点位。

6、更多更详细的设置可以查看官方文档。
